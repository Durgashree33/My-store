My E-Commerce Product Page

A simple, interactive e-commerce product page built using HTML, CSS, and JavaScript. Users can view products and add them to a shopping cart.
Features

    Displays products dynamically on the page
    Add products to the shopping cart with a click
    Shows total number of items in the cart
    Responsive design for both mobile and desktop

Technologies Used

    HTML – for page structure
    CSS – for styling and layout
    JavaScript – for interactivity, like adding items to the cart
