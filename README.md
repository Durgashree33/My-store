# 🛒 My-Store | Responsive E-Commerce Product Page

**Interactive online store project** built with **HTML, CSS, and JavaScript**.  

This project demonstrates a simple e-commerce product page where users can **view products, add items to a shopping cart**, and see the **total number of items dynamically updated**.  

---

# 🎯 Overview
A minimal, interactive e-commerce page with:
- Dynamic product display on the page  
- Add items to the shopping cart with a click  
- Real-time cart item count  
- Fully responsive design for mobile and desktop  
- Clean and modern user interface  

---

# ⚙️ Technologies Used
- **HTML5** – Page structure and layout  
- **CSS3** – Styling, animations, and responsive design  
- **JavaScript** – Dynamic functionality (cart interactions, updating counts)  

---

# 🕹️ Features
- 🛍️ Product listing displayed dynamically  
- ➕ Add products to shopping cart  
- 📊 Real-time cart counter  
- 📱 Mobile-first responsive design  
- 🎨 Clean UI and smooth interactions  

---
